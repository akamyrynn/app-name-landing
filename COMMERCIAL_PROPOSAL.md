# Коммерческое предложение / Оценка трудозатрат

**Проект:** Онлайн-конфигуратор столешниц с 3D визуализацией и CRM.
**Исполнитель:** AI Assistant (Architecture & Code).

Данный документ содержит предварительную оценку времени на доработку текущего прототипа до полноценного релиза (Production Ready).

---

## Смета работ (в часах разработки)

### 1. Доработка Конфигуратора (Frontend & 3D)
*   **Интеграция UI из Figma:** Верстка финального дизайна, анимации, адаптив под мобильные устройства.
    *   *Оценка: 20-30 часов.*
*   **Логика валидации:** Ограничения на вырезы, проверка минимальных/максимальных размеров, конфликт вырезов.
    *   *Оценка: 8-12 часов.*
*   **Оптимизация 3D:** Загрузка текстур, работа со светом, "запекание" теней для реалистичности.
    *   *Оценка: 10-15 часов.*

### 2. Личный кабинет и Авторизация
*   **Система входа:** Реализация входа по SMS/Телефону (Supabase Auth).
    *   *Оценка: 6-8 часов.*
*   **Страница статуса:** Верстка и лоигка отображения статусов заказа.
    *   *Оценка: 5-8 часов.*

### 3. Бэкенд и Интеграции
*   **Telegram Бот:** Создание бота для уведомлений менеджеров.
    *   *Оценка: 4-6 часов.*
*   **Платежная система:** Интеграция эквайринга (API банка/агрегатора).
    *   *Оценка: 8-12 часов.*
*   **Настройка сервера:** DNS, SSL, домены, Supabase Security Policies (RLS).
    *   *Оценка: 4-6 часов.*

### 4. Тестирование и Отладка
*   Кроссбраузерное тестирование, исправление багов, проверка PDF генерации.
    *   *Оценка: 10-15 часов.*

## Необходимые ресурсы от Клиента
Для начала активной фазы доработки требуются:
1.  **Figma макеты:** Финальный дизайн всех экранов (Desktop + Mobile).
2.  **Текстуры:** Архивы качественных текстур (Color, Normal, Roughness карты).
3.  **Доступы:** Аккаунт Supabase (если меняется), доступы к домену.
4.  **Платежный шлюз:** Выбор провайдера и получение API ключей.

## План на ближайшую неделю
1.  Получение дизайна и утверждение ТЗ.
2.  Настройка авторизации (Личный кабинет).
3.  Верстка по макетам.
