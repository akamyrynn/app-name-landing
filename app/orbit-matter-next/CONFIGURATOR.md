# Интерактивный Конфигуратор Столешниц

## Функционал

### 1. Выбор материала
- **Дерево**: Дуб, Орех
- **Камень**: Мрамор, Гранит  
- **Металл**: Сталь, Латунь
- Возможность добавления новых материалов через `pageData.ts`

### 2. Настройка размеров
- **Длина**: 60-300 см (слайдер)
- **Ширина**: 40-200 см (слайдер)
- **Толщина**: 2-10 см (слайдер)
- Реалтайм обновление 3D модели

### 3. Выбор формы столешницы
- Прямоугольная
- Круглая
- Трапеция
- Нестандартная (custom)

### 4. Добавление вырезов
- Мойка
- Варочная панель
- Решётка
- Техническое отверстие
- Управление списком вырезов (добавление/удаление)

### 5. 3D Визуализация
- Реалистичное отображение материалов (PBR)
- Вращение модели (OrbitControls)
- Масштабирование
- Корректные пропорции
- Освещение студийного качества
- Тени и отражения

### 6. Экспорт результата
- Кнопка "Экспорт в PDF" (готова к интеграции)
- Кнопка "Добавить в корзину" (готова к интеграции)

## Дизайн

### Левая панель (Темная карточка)
- Заголовок и описание
- 3D Canvas с интерактивной моделью
- Информационная панель с текущими параметрами

### Правая панель (Светлая карточка)
- Бежевый фон (#F2EEDA)
- Скроллируемая область с контролами
- Карточки выбора материалов с превью
- Слайдеры для размеров
- Кнопки управления вырезами
- Основные действия (корзина, экспорт)

## Технологии
- React Three Fiber
- Three.js
- @react-three/drei (OrbitControls, Environment)
- GSAP (анимации)
- Next.js 14

## Адаптивность
- Desktop: двухколоночный layout
- Tablet/Mobile: вертикальный стек
- Оптимизированные размеры для всех экранов
